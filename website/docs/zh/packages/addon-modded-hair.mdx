# @scml/addon-modded-hair

Modded hair addon 的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/addon-modded-hair" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/addon-modded-hair" />
```

主入口增强以下全局属性：`addonModdedHairAddon`（`ModdedHairAddon` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/addon-modded-hair/ModdedHairAddon';
```

## 导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口，增强 Window/全局 |
| `./ModdedHairAddon` | 类型声明 |
| `./ModdedHairAddonParams` | 类型声明 |

## 类型

### 主入口
- **`addonModdedHairAddon`** – 添加并合并 modded 发型数据的 Addon。

### `./ModdedHairAddon`
- **`ModdedHairAddon`** – 持有 `hairData: Map<string, HairObject>` 的 Addon 插件。方法：`registerMod()`、`exportDataZip()`、`init()`。
- **`mergeHairObject()`**、**`appendHairObject()`** – 合并/追加发型工具。

### `./ModdedHairAddonParams`
- Addon 参数配置；详见子路径类型定义。
