# @scml/mod-check-dol-compressor-dictionaries

DoL compressor 词典检测的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/mod-check-dol-compressor-dictionaries" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/mod-check-dol-compressor-dictionaries" />
```

主入口增强以下全局属性：`modCheckDoLCompressorDictionaries`（`CheckDoLCompressorDictionaries` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/mod-check-dol-compressor-dictionaries/CheckDoLCompressorDictionaries';
```

## 导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口，增强 Window/全局 |
| `./CheckDoLCompressorDictionaries` | 类型声明 |

## 类型

### 主入口
- **`modCheckDoLCompressorDictionaries`** – 检测 DoL compressor 词典兼容性。

### `./CheckDoLCompressorDictionaries`
- **`CheckDoLCompressorDictionaries`** – 验证 compressor 词典的 Addon 类。
