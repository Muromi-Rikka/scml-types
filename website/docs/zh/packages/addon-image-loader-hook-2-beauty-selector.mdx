# @scml/addon-image-loader-hook-2-beauty-selector

ImageLoader Hook 2 Beauty Selector addon 的 TypeScript 类型定义，桥接 ImgLoader 与 BeautySelector，支持图片转换与选择器集成。

## 安装

<PackageManagerTabs command="install -D @scml/addon-image-loader-hook-2-beauty-selector" />

## 快速开始

<Steps>

### 添加类型引用

```ts
/// <reference types="@scml/addon-image-loader-hook-2-beauty-selector" />
```

### 使用全局实例

`modImageLoaderHook2BeautySelectorAddon` 已挂载到 `window` 上。

### 按需导入类型

```ts
import type { ImageLoaderHook2BeautySelectorAddon, AddonParams } from '@scml/addon-image-loader-hook-2-beauty-selector/ImageLoaderHook2BeautySelectorAddon';
```

</Steps>

## API 参考

### ./ImageLoaderHook2BeautySelectorAddon

#### ImageLoaderHook2BeautySelectorAddon <Badge text="类" />

| 属性 | 类型 | 说明 |
|------|------|------|
| `modImageLoaderHookRef` | `any` | ImageLoader Hook 引用 |
| `nameSetImageLoaderHook` | `Set<string>` | 已注册的 ImageLoader Hook 名称 |
| `convertedModeNameList` | `string[]` | 已转换的 mod 名列表 |
| `registerMod()` | `(addonName, mod, modZip) => Promise<any>` | 注册 mod |
| `canLoadThisMod()` | `(bootJson, zip) => Promise<boolean>` | 是否可加载该 mod |
| `ModLoaderLoadEnd()` | `() => Promise<void>` | Mod Loader 加载完成回调 |

#### AddonParams <Badge text="接口" />

| 属性 | 类型 | 说明 |
|------|------|------|
| `dontCovertMe` | `boolean` | 不转换该 mod |
