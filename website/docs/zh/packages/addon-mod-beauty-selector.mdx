# @scml/addon-mod-beauty-selector

Beauty Selector addon 的 TypeScript 类型定义，用于图片选择器 UI 与 mod 图片管理。

## 安装

<PackageManagerTabs command="install -D @scml/addon-mod-beauty-selector" />

## 快速开始

<Steps>

### 添加类型引用

```ts
/// <reference types="@scml/addon-mod-beauty-selector" />
```

### 使用全局实例

`addonBeautySelectorAddon` 已挂载到 `window` 上。

### 按需导入类型

```ts
import type { BeautySelectorAddon } from '@scml/addon-mod-beauty-selector/BeautySelectorAddon';
import type { BeautySelectorAddonInterface } from '@scml/addon-mod-beauty-selector/BeautySelectorAddonInterface';
```

</Steps>

## API 参考

### ./BeautySelectorAddon

Beauty Selector 主 Addon 类，实现 `BeautySelectorAddonInterface` 与 `AddonPluginHookPointEx`，提供图片选择器与 mod 图片管理功能。具体 API 见 `BeautySelectorAddonInterface` 及类型定义。
