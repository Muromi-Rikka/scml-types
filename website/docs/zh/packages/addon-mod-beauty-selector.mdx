# @scml/addon-mod-beauty-selector

Beauty selector addon 的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/addon-mod-beauty-selector" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/addon-mod-beauty-selector" />
```

主入口增强以下全局属性：`addonBeautySelectorAddon`（`BeautySelectorAddon` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/addon-mod-beauty-selector/BeautySelectorAddon';
```

## 类型

### 主入口
- **`addonBeautySelectorAddon`** – 与 Sub UI AngularJS 及 Mod Image Store 集成的 Beauty selector addon。

### `./BeautySelectorAddon`
- **`BeautySelectorAddon`** – 主 Addon 类；管理 beauty 选项、参数及 AngularJS 组件。

### `./BeautySelectorAddonInterface`
- Beauty selector addon 行为的接口。

### `./BeautySelectorAddonType`
- **`BeautySelectorAddonParamsType0`**、**`Type1`**、**`Type2`** – 参数配置类型。
- **`TypeOrderItem`**、**`BeautySelectorAddonParamsType2ATypeItem`**、**`Type2BTypeItem`** – 选项项类型。

### `./ModImageStore`
- **`ModImageStore`** – mod 资源的图片存储。
- **`CachedFileList`** – 缓存文件列表类型。

### `./extname` / `./utils/*` / `./GUI_StringTable/*`
- 工具类型及 i18n 字符串表。
