# @scml/hook-macro-rng

Macro RNG hook 的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/hook-macro-rng" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/hook-macro-rng" />
```

主入口增强以下全局属性：`hookMacroRng`（`HookMacroRng` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/hook-macro-rng/HookMacroRng';
```

## 导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口，增强 Window/全局 |
| `./HookMacroRng` | 类型声明 |

## 类型

### 主入口
- **`hookMacroRng`** – 挂载 SugarCube 宏使用的 RNG，以实现确定性或自定义 RNG 行为。

### `./HookMacroRng`
- **`HookMacroRng`** – 安装 macro RNG hook 的 Addon 类（从 `inject_early.d.ts` 导出）。
