# @scml/hook-macro-rng

Macro RNG hook 的 TypeScript 类型定义，挂载 SugarCube 宏使用的 RNG，以实现确定性或自定义 RNG 行为。

## 安装

<PackageManagerTabs command="install -D @scml/hook-macro-rng" />

## 快速开始

<Steps>

### 添加类型引用

```ts
/// <reference types="@scml/hook-macro-rng" />
```

### 使用全局实例

`hookMacroRng` 已挂载到 `window` 上。

</Steps>

## 全局类型 <Badge type="info" text="global" />

| 属性 | 类型 | 说明 |
|------|------|------|
| `hookMacroRng` | `HookMacroRng` | Macro RNG Hook 实例 |
