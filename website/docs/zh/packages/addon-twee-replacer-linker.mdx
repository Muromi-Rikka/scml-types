# @scml/addon-twee-replacer-linker

Twee replacer linker addon 的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/addon-twee-replacer-linker" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/addon-twee-replacer-linker" />
```

主入口增强以下全局属性：`addonTweeReplacerLinker`（`TweeReplacerLinker` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/addon-twee-replacer-linker/TweeReplacerLinker';
```

## 导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口，增强 Window/全局 |
| `./TweeReplacerLinker` | 类型声明 |
| `./TweeReplacerLinkerInterface` | 类型声明 |

## 类型

### 主入口
- **`addonTweeReplacerLinker`** – 连接 TweeReplacer 客户端；协调跨 mod 的替换回调。

### `./TweeReplacerLinker`
- **`TweeReplacerLinker`** – 实现 `TweeReplacerLinkerInterface` 的 Addon。方法：`registerClient()`、`addUserMod()`、`registerMod()`、`afterPatchModToGame()`。
- **`TweeReplacerLinkerCallbackData`** – 回调注册数据。

### `./TweeReplacerLinkerInterface`
- **`TweeReplacerLinkerInterface`** – `registerClient()`、`addUserMod()`。
- **`TweeReplacerLinkerClientInterface`** – `enableLinkerMode()`。
- **`TweeReplacerLinkerClientCallbackType`** – `(sc: SC2DataInfo) => Promise<any>`。
