# @scml/mod-replacer-patch

Replace patch 的 TypeScript 类型定义，对游戏/mod 内容执行查找替换补丁。

## 安装

<PackageManagerTabs command="install -D @scml/mod-replacer-patch" />

## 快速开始

<Steps>

### 添加类型引用

```ts
/// <reference types="@scml/mod-replacer-patch" />
```

### 使用全局实例

`addonReplacePatcher` 已挂载到 `window` 上。

</Steps>

## 全局类型 <Badge type="info" text="global" />

| 属性 | 类型 | 说明 |
|------|------|------|
| `addonReplacePatcher` | `ReplacePatcher` | 替换补丁 Addon 实例 |

## API 参考

#### ReplacePatcher <Badge text="类" />

应用替换补丁的 Addon 类，对 js/css/twee 内容进行查找/替换。
