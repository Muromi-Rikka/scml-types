# @scml/addon-mod-i18n-twee-list

i18n Twee list addon 的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/addon-mod-i18n-twee-list" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/addon-mod-i18n-twee-list" />
```

主入口增强以下全局属性：`modI18nTweeList`（`I18nTweeList` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/addon-mod-i18n-twee-list/I18nTweeList';
```

## 导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口，增强 Window/全局 |
| `./I18nTweeList` | 类型声明 |

## 类型

### 主入口
- **`modI18nTweeList`** – 管理 mod 的 i18n Twee 列表的 Addon。

### `./I18nTweeList`
- **`I18nTweeList`** – 注册和管理 i18n Twee 列表数据的 Addon 类。

## 类型定义来源

- 构建: `pnpm run download && pnpm run build`
