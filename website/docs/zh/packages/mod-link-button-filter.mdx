# @scml/mod-link-button-filter

Link button filter 的 TypeScript 类型定义，修改 DoL 中链接按钮行为。

## 安装

<PackageManagerTabs command="install -D @scml/mod-link-button-filter" />

## 快速开始

<Steps>

### 添加类型引用

```ts
/// <reference types="@scml/mod-link-button-filter" />
```

### 使用全局函数

`DoLLinkButtonFilter_patchLinkButton` 已挂载到 `window` 上。

</Steps>

## 全局类型 <Badge type="info" text="global" />

| 属性 | 类型 | 说明 |
|------|------|------|
| `DoLLinkButtonFilter_patchLinkButton` | `typeof patchLinkButton` | 修改链接按钮行为，过滤或修改链接点击 |
