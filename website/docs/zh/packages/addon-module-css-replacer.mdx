# @scml/addon-module-css-replacer

Module CSS replacer addon 的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/addon-module-css-replacer" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/addon-module-css-replacer" />
```

主入口增强以下全局属性：`addonModuleCssReplacer`（`ModuleCssReplacer` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/addon-module-css-replacer/ModuleCssReplacer';
```

## 导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口，增强 Window/全局 |
| `./ModuleCssReplacer` | 类型声明 |

## 类型

### 主入口
- **`addonModuleCssReplacer`** – `ModuleCssReplacer` 实例；模块 CSS 文件的替换 addon。

### `./ModuleCssReplacer`
- **`ModuleCssReplacer`** – 替换模块 CSS 中内容的 Addon 插件。暴露 `ReplaceParams`（如 `cssName`、`findString`、`findRegex`、`replace`）、`registerMod()`、`do_patch()`、`getAllModuleStyleNode()`。
