# @scml/mod-sub-ui-angular-js

Sub UI (AngularJS) 的 TypeScript 类型定义，基于 AngularJS 的子 UI 管理器。

## 安装

<PackageManagerTabs command="install -D @scml/mod-sub-ui-angular-js" />

## 快速开始

<Steps>

### 添加类型引用

```ts
/// <reference types="@scml/mod-sub-ui-angular-js" />
```

### 使用全局实例

主入口增强 `diff3WayMerge`（`ModSubUiAngularJs`）。

</Steps>

## 全局类型 <Badge type="info" text="global" />

| 属性            | 类型                | 说明                   |
| --------------- | ------------------- | ---------------------- |
| `diff3WayMerge` | `ModSubUiAngularJs` | 子 UI AngularJS 管理器 |

## 子路径导出

| 子路径                                   | 描述               |
| ---------------------------------------- | ------------------ |
| `.`                                      | 主入口             |
| `./ModSubUiAngularJs`                    | 主类               |
| `./ModSubUiAngularJsBody`                | body 类型          |
| `./AppContainerManagerMethodsInterface`  | 应用容器管理接口   |
| `./ModSubUiAngularJsModeExportInterface` | 模式导出接口       |
| `./AngularJs/appInit`                    | Angular 应用初始化 |
| `./AngularJs/ExternalComponentManager`   | 外部组件管理器     |
| `./AngularJs/Componnet/*`                | 各类 UI 组件       |

## API 参考

### AppContainerManagerMethodsInterface

针对 `ModGuiConfig` 与 `ModInGameConfig` 的 bootstrap/release/addComponent/cleanComponent/registryComponent。
