# @scml/mod-diff3-way-merge

Three-way merge 的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/mod-diff3-way-merge" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/mod-diff3-way-merge" />
```

主入口增强以下全局属性：`diff3WayMerge`（`Diff3WayMerge` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/mod-diff3-way-merge/Diff3WayMerge';
```

## 导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口，增强 Window/全局 |
| `./Diff3WayMerge` | 类型声明 |
| `./Diff3WayMergeParams` | 类型声明 |
| `./DiffFormat` | 类型声明 |
| `./diff_match_patch/diff_match_patch` | 类型声明 |

## 类型

### 主入口
- **`diff3WayMerge`** – 对 mod 补丁执行三路合并（base + ours + theirs）。

### `./Diff3WayMerge`
- **`Diff3WayMerge`** – 持有 `info: Map<string, ModMergeInfo>` 的 Addon。方法：`registerMod()`、`afterPatchModToGame()`、`do_patch()`、`readOriginFile()`、`readModFile()`、`readDiffFormatFile()`。
- **`ModMergeInfo`** – `{ addonName, mod, modZip }`。

### `./Diff3WayMergeParams` / `./DiffFormat` / `./diff_match_patch/diff_match_patch`
- 合并参数、diff 格式类型及 diff-match-patch 库绑定。
