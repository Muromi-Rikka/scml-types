# @scml/hook-img-loader

Image loader hook 的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/hook-img-loader" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/hook-img-loader" />
```

主入口增强以下全局属性：`modImgLoaderHooker`（`ImgLoaderHooker` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/hook-img-loader/ImgLoaderHooker';
```

## 类型

### 主入口
- **`modImgLoaderHooker`** – 挂载图片加载器，允许 mod 提供自定义图片。

### `./ImgLoaderHooker`
- **`ImgLoaderHooker`** – 继承 `ImgLoaderHookerCore`；安装 hook 并等待初始化。方法：`init()`、`installHook()`。

### `./ImgLoaderHookerCore` / `./NodeMutationObserver` / `./SimpleDolFunctionHook`
- 核心 hook 基础设施及 mutation/函数 hook 工具。

### `./CssReplacer` / `./utils` / `./createFunctionFromCode`
- CSS 替换辅助、工具及从代码生成函数的工厂。
