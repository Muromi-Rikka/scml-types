# @scml/mod-i18n

DoL i18n mod 的 TypeScript 类型定义，处理翻译与本地化数据。

## 安装

<PackageManagerTabs command="install -D @scml/mod-i18n" />

## 快速开始

<Steps>

### 添加类型引用

```ts
/// <reference types="@scml/mod-i18n" />
```

### 使用全局实例

`modI18N` 已挂载到 `window` 上。

### 按需导入类型

```ts
import type { ModI18N } from '@scml/mod-i18n/I18N';
import type { GetPromiseWithResolvers } from '@scml/mod-i18n/GetPromiseWithResolvers';
```

</Steps>

## 子路径导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口，增强 Window/全局 |
| `./I18N` | ModI18N 主类 |
| `./GetPromiseWithResolvers` | Promise 解析器辅助 |
| `./TypeB` | i18n 输入/输出类型 |

## API 参考

### ./I18N

#### ModI18N <Badge text="类" />

主 i18n 类。属性：`modUtils`、`modSC2DataManager`、`logger`、`idbRef`、`idbKeyValRef`、`typeB`。方法：`patchVersionString()`、`readZipStream()`、`readZipSelf()`、`checkItem()` 等。
