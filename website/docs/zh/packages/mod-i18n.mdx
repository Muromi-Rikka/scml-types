# @scml/mod-i18n

DoL i18n mod 的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/mod-i18n" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/mod-i18n" />
```

主入口增强以下全局属性：`modI18N`（`ModI18N` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/mod-i18n/I18N';
```

## 导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口，增强 Window/全局 |
| `./GetPromiseWithResolvers` | 类型声明 |
| `./I18N` | 类型声明 |
| `./TypeB` | 类型声明 |

## 类型

### 主入口
- **`modI18N`** – DoL i18n mod 实例；处理翻译与本地化数据。

### `./I18N`
- **`ModI18N`** – 主 i18n 类。属性：`modUtils`、`modSC2DataManager`、`logger`、`idbRef`、`idbKeyValRef`、`typeB`。方法：`patchVersionString()`、`readZipStream()`、`readZipSelf()`、`checkItem()`。
- **`sleep()`** – `Promise` 延迟工具。

### `./GetPromiseWithResolvers` / `./TypeB`
- **`GetPromiseWithResolvers`** – Promise 解析器辅助。
- **`TypeB`** – i18n 输入/输出类型定义，及 `ModI18NTypeB`、`TypeBI18NInputType`。
