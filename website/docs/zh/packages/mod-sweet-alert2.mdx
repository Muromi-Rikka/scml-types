# @scml/mod-sweet-alert2

SweetAlert2 mod 的 TypeScript 类型定义，封装 SweetAlert2 供 SugarCube mod 使用。

## 安装

<PackageManagerTabs command="install -D @scml/mod-sweet-alert2" />

## 快速开始

<Steps>

### 添加类型引用

```ts
/// <reference types="@scml/mod-sweet-alert2" />
```

### 使用全局实例

`modSweetAlert2Mod` 已挂载到 `window` 上。

</Steps>

## 全局类型 <Badge type="info" text="global" />

| 属性 | 类型 | 说明 |
|------|------|------|
| `modSweetAlert2Mod` | `SweetAlert2Mod` | SweetAlert2 mod 实例 |

## 子路径导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口 |
| `./SweetAlert2Mod` | 主类 |

## API 参考

#### SweetAlert2Mod <Badge text="类" />

提供 `fire`（`typeof Swal.fire`）、`fireWithOptions(options)` 及 `Swal`（`typeof SweetAlert2`）。
