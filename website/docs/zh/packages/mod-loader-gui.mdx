# @scml/mod-loader-gui

Mod Loader GUI 的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/mod-loader-gui" />

## 快速开始

<Steps>

### 添加类型引用

```ts
/// <reference types="@scml/mod-loader-gui" />
```

### 使用全局实例

主入口增强以下全局属性。

</Steps>

## 全局类型 <Badge type="info" text="global" />

| 属性 | 类型 | 说明 |
|------|------|------|
| `modLoaderGui_LoadingProgress` | - | 加载进度 UI 组件 |
| `modLoaderGui_PassageTracer` | - | 段落追踪 |
| `modLoaderGui` | `Gui` | 主 GUI 控制器 |
| `modLoaderGui_ModSubUiAngularJsService` | - | mod 子 UI AngularJS 服务 |

## 子路径导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口 |
| `./Gui` | 主 GUI 类 |
| `./LoadingProgress` | 加载进度 |
| `./PassageTracer` | 段落追踪 |
| `./ModLoadSwitch` | Mod 加载开关 |
| `./ModManagerSubUi` | Mod 管理器 UI |
| `./ModSubUiAngularJsService` | AngularJS 服务 |
| `./DebugExport` | 调试工具 |
| `./GUI_StringTable/*` | 字符串表 |
| `./GM_config_TS/*` | GM_config 类型 |

## API 参考

### ./Gui

主 GUI 类，控制 Mod Loader 界面。
