# @scml/mod-i18n-twee-replacer

i18n Twee replacer 的 TypeScript 类型定义，将 i18n 翻译合并到 Twee 内容中。

## 安装

<PackageManagerTabs command="install -D @scml/mod-i18n-twee-replacer" />

## 快速开始

<Steps>

### 添加类型引用

```ts
/// <reference types="@scml/mod-i18n-twee-replacer" />
```

### 使用全局实例

`modI18nTweeList` 已挂载到 `window` 上。

</Steps>

## 全局类型 <Badge type="info" text="global" />

| 属性              | 类型           | 说明                    |
| ----------------- | -------------- | ----------------------- |
| `modI18nTweeList` | `I18nTweeList` | i18n Twee replacer 实例 |
