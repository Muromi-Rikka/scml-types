# @scml/addon-mod-i18n-script-list

i18n script list addon 的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/addon-mod-i18n-script-list" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/addon-mod-i18n-script-list" />
```

主入口增强以下全局属性：`modI18nScriptList`（`I18nScriptList` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/addon-mod-i18n-script-list/I18nScriptList';
```

## 导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口，增强 Window/全局 |
| `./I18nScriptList` | 类型声明 |

## 类型

### 主入口
- **`modI18nScriptList`** – 管理 mod 的 i18n 脚本列表的 Addon。

### `./I18nScriptList`
- **`I18nScriptList`** – 注册和管理 i18n 脚本列表数据的 Addon 类。
