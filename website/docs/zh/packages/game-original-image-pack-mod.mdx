# @scml/game-original-image-pack-mod

游戏原始图片包的 TypeScript 类型定义。

## 安装

<PackageManagerTabs command="install -D @scml/game-original-image-pack-mod" />

## 用法

### 主入口（全局类型）

```ts
/// <reference types="@scml/game-original-image-pack-mod" />
```

主入口增强以下全局属性：`modGameOriginalImagePack`（`GameOriginalImagePack` 实例）。

### 子路径导入

```ts
import type { SomeType } from '@scml/game-original-image-pack-mod/GameOriginalImagePack';
```

## 导出

| 子路径 | 描述 |
|--------|------|
| `.` | 主入口，增强 Window/全局 |
| `./GameOriginalImagePack` | 类型声明 |

## 类型

### 主入口
- **`modGameOriginalImagePack`** – 提供对游戏原始图片包的访问。

### `./GameOriginalImagePack`
- **`GameOriginalImagePack`** – 加载并暴露游戏原始图片的 Mod 类。
