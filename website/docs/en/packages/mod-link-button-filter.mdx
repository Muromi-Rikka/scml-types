# @scml/mod-link-button-filter

TypeScript type definitions for Link button filter, modifying link button behavior in DoL.

## Installation

<PackageManagerTabs command="install -D @scml/mod-link-button-filter" />

## Quick Start

<Steps>

### Add type reference

```ts
/// <reference types="@scml/mod-link-button-filter" />
```

### Use global function

`DoLLinkButtonFilter_patchLinkButton` is mounted on `window`.

</Steps>

## Global Types <Badge type="info" text="global" />

| Property                              | Type                     | Description                                                    |
| ------------------------------------- | ------------------------ | -------------------------------------------------------------- |
| `DoLLinkButtonFilter_patchLinkButton` | `typeof patchLinkButton` | Modifies link button behavior; filters or modifies link clicks |
