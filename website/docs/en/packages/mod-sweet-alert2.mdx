# @scml/mod-sweet-alert2

TypeScript type definitions for SweetAlert2 mod, wrapping SweetAlert2 for SugarCube mod use.

## Installation

<PackageManagerTabs command="install -D @scml/mod-sweet-alert2" />

## Quick Start

<Steps>

### Add type reference

```ts
/// <reference types="@scml/mod-sweet-alert2" />
```

### Use global instance

`modSweetAlert2Mod` is mounted on `window`.

</Steps>

## Global Types <Badge type="info" text="global" />

| Property | Type | Description |
|----------|------|-------------|
| `modSweetAlert2Mod` | `SweetAlert2Mod` | SweetAlert2 mod instance |

## Subpath Exports

| Subpath | Description |
|---------|-------------|
| `.` | Main entry |

## API Reference

#### SweetAlert2Mod <Badge text="class" />

Provides `fire` (`typeof Swal.fire`), `fireWithOptions(options)`, and `Swal` (`typeof SweetAlert2`).
