# @scml/mod-sub-ui-angular-js

TypeScript type definitions for Sub UI (AngularJS), AngularJS-based sub UI manager.

## Installation

<PackageManagerTabs command="install -D @scml/mod-sub-ui-angular-js" />

## Quick Start

<Steps>

### Add type reference

```ts
/// <reference types="@scml/mod-sub-ui-angular-js" />
```

### Use global instance

Main entry augments `diff3WayMerge` (`ModSubUiAngularJs`).

</Steps>
