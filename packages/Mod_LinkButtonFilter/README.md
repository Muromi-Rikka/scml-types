# @scml/mod-link-button-filter

TypeScript type definitions for Link button filter.

## Installation

```bash
pnpm add -D @scml/mod-link-button-filter
```

## Usage

### Main entry (global types)

```ts
/// <reference types="@scml/mod-link-button-filter" />
```

Main entry augments these global properties: `DoLLinkButtonFilter_patchLinkButton`

## Exports

| Subpath | Description |
|--------|------|
| `.` | Main entry, augments Window/global |

## Type definition source

- Upstream: https://github.com/Lyoko-Jeremie/DoLLinkButtonFilter
- Default branch: master
- Build: `pnpm run download && pnpm run build`
